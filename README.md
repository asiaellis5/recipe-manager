# recipe-manager

## User Stories

```
As a user,
So I can see the recipes that I have saved,
I want to be able to see a list of them

As a user,
So that I can save new recipes,
I want to be able to add them to the list

As a user,
So that I can amend recipes that I have in my list,
I want to be able to update them

As a user,
So that I can get rid of any recipes that I don't want,
I want to be able to delete them
```
